# Create Repo README

This skill generates a comprehensive `README.md` for a software repository, focusing on clear project details and usage instructions.

## Usage

1.  Ask the skill: "@create-repo-readme" or "create a readme for [PROJECT_NAME]".
2.  Provide brief details about the project (Name, Description, Tech Stack).
3.  The skill will generate a structured README with sections like:
    -   Project Title & Description
    -   Key Features
    -   Installation
    -   Usage
    -   Contributing
    -   License
