# Create LinkedIn Posts

This skill generates catchy and clickbaity LinkedIn posts based on a user-provided topic.

## Usage

1.  Ask the skill: "@create-linkedin-posts [TOPIC]" or "I need a LinkedIn post about [TOPIC]".
2.  Provide the topic you want the posts to be about.
3.  The skill will respond with 2-3 distinct post options.
4.  Choose the best one or ask for revisions.
